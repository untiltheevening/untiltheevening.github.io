0) Purchased a domain name @ iwantmyname

1) Set up the cheapest droplet I could (5 dollar/mo)

2) Directed the domain name to DNS (understand this better)


3) Install nginx

sudo apt-get update
sudo apt-get install nginx


4) Set up a directory for the website
sudo mkdir /srv/www /srv/www/untiltheevening.com

Configure nginx to serve files from that directory.
Open ```/etc/nginx/sites-available/default```
and change ```root XYZ``` to ```root /srv/www/untiltheevening.com```

Restart nginx. ```sudo service nginx restart```

Give myself write access to directory
```sudo chown -R $USER /srv/www```
